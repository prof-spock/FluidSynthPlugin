Boolean _MidiEventConverterDescriptor::handleProgramChange ( IN String &  value,
IN Boolean  changeIsForced = false 
)

Handles bank and program change depending on for given string value on synthesizer.

value contains bank and program number separated by a colon. The bank number comes first followed by a colon and finally the program number. All MIDI channels are affected, if the bank is missing, bank number 0 is assumed; an empty bank is ignored and an empty program is erroneous.

Parameters
[in]valuestring value for bank and/or program
[in]changeIsForcedtells whether program change must be done unconditionally
Returns
information whether program change handling has been successful
