Detailed Description

The package BaseTypes provides elementary types like primitives and containers.

Sub-package Containers specifies and implements dictionaries, maps from string to objects, lists of string or numbers and string sets.

In sub-package GenericTypes generic types are defined providing numbers and enumerations for primitives and lists, maps, tuples and matrices for containers.

In sub-package Primitives elementary types of C++ like ints and chars are wrapped into classes like Integer, Natural, String etc. This approach allows a more consistent program documentation.
