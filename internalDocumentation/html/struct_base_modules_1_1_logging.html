Detailed Description

A Logging object defines methods for entry/exit logging to a file or via a callback function; this logging relies on trace calls at the beginning or end of a function with prefices ">>" and "<<" as well as intermediate log lines with prefix "--"; the name of the function is also logged to give a fully bracketed log.
